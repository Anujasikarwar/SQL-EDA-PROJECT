## Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a global layoffs dataset using SQL. The goal is to analyze trends, identify patterns, and generate business insights from the cleaned data.

## Dataset
- Dataset: Global Layoffs Dataset
- Source: Kaggle
- The dataset was cleaned before performing exploratory analysis.

## Objectives
- Analyze total layoffs by company
- Analyze layoffs by industry
- Analyze layoffs by country
- Analyze layoffs over time
- Identify companies with the highest layoffs
- Find trends using SQL queries

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- Window Functions
- Common Table Expressions (CTEs)
- Date Functions

## Files
- eda_project.sql – SQL queries for exploratory data analysis.
- layoffs.csv – Dataset used in the project.

## Key Insights
- Identified companies with the highest layoffs.
- Analyzed layoffs by country and industry.
- Observed layoff trends over different time periods.
- Ranked companies based on total layoffs.
